Alternative appeals journey

Alternative 1

An alternative journey for the appeals process.

  • User uses find function on workstack and is presented with results.
  • User selects the result they require.
  • Once selected the user is taken to a summary page where they can view case details and select to log an appeal against this case which would beging the current appeals process.
    • (b) This step could be removed and the user could be taken into the the appeals process.

This would not work for live cases i.e. time based appeals would work for closed cases.
